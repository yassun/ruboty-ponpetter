# Ruboty::Ponpetter

Ruboty handler to show today ponpe count.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-ponpetter'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-ponpetter

## Usage

```
@ruboty ponpetter
只今のポンペ数は17174人です。

```
